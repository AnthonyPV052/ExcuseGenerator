# ExcuseGenerator

Un generador simple de excusas aleatorias escrito en JavaScript. Proyecto de PreWork.

## Descripción

Este proyecto genera excusas divertidas y aleatorias combinando elementos de listas predefinidas: quién, qué acción, qué objeto y cuándo. Es útil para situaciones en las que necesitas una excusa rápida y creativa.

## Requisitos

- Node.js instalado en tu sistema.

## Uso

1. Clona o descarga el repositorio.
2. Navega al directorio del proyecto.
3. Ejecuta el script con Node.js:


Cada vez que ejecutes el script, se generará una nueva excusa aleatoria y se imprimirá en la consola.

## Ejemplo de salida

```
Mi perro se comió mi tarea anoche.
```

## Personalización

Puedes modificar las listas de elementos en `excuse-generator.js` para agregar más opciones o cambiar el idioma:

- `who`: Lista de sujetos (ej. "Mi perro", "Mi gato").
- `action`: Lista de acciones (ej. "se comió", "rompió").
- `what`: Lista de objetos (ej. "mi tarea", "mi teléfono").
- `when`: Lista de tiempos (ej. "hoy", "ayer").
